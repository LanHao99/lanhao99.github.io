### 网上下载资源的方法

1. 寻找开源素材

打开设置好的魔法，我们可以在一些公开的开源视频网站找可用视频

我最常用的是pixabay

2. bt种子下载

如果你想要下载无字幕的动漫或者电影素材，你就需要学会找bt种子并下载。

bt种子可以理解为一个蓝图清单，会告诉程序要去哪里下载什么东西。bt种子的下载是你从其他人电脑里下好的文件里传输过来，所以说，下载的人越多，越新的种子，传输速度越快。那些老种子很可能每人上传了，所以下载的就很慢。这时候就只能找找其他的歪门邪道了……

bt种子站有很多，我也分享在网盘里了，自己找

接下来是下载种子

很多网盘软件都有bt种子下载功能，不过由于它们为了搞钱经常会破坏bt种子，因此很多网站禁止你用网盘下载bt资源。

你需要一个合规的bt种子下载器。我一般使用qbittorrent。下载包都在网盘里面。

另外，现在很多上传者顺便会挂一个网盘链接，那么直接用网盘链接下载就行了。

如果你想要长期自动下载当季新番，可以使用rss订阅

3. 浏览器工具

很多浏览器工具可以一键下载网上内容，一般配合一个素材管理软件会更方便。

billfish免费，eagle付费，各有千秋，因为他们没给我米，所以就不在这里详细介绍了，感兴趣的b站上有很多介绍。

4. yt-dlp

可以通过一个开源程序YT DLP下载youtube视频，不过这个方法最近不是很稳定。

直接让AI帮忙操作就行了

5. 下载歌曲，提取歌词

免费下歌的方法其实有很多，比如说我现在在用S player，一个开源软件，它会把音频缓存保存到一个地方，我们找到缓存的音频，把文件尾缀改成mp3就行了。

### 程序自动切分视频分镜

从网上下载好了视频之后，我们会面临一个新的问题：

这么多视频，怎么找到我想要的资源？

对于一个二十多分钟长的动漫，我们往往只要其中一段内容，这样的话，我们就需要给视频

我一般用达芬奇剪辑，现在的剪辑软件都有自动分镜功能，达芬奇可以在媒体池选择“自动剪切探测”，但是这样还得手动一个一个操作，很麻烦，有没有自动探测分镜的方法呢？

有的兄弟有的，使用python的一个库可以自动剪切视频

首先最难的一步，安装依赖。你可以通过AI编程软件来辅助操作，比如trae、cursor或者claude code。

运行pip install scenedetect​，下载python库。

这个库还需要处理视频的依赖软件ffmpeg，所以你还得安装ffmpeg。这个网盘里也有，自己安装即可。

然后让你的AI给你写一个自动剪切视频的python文件即可。可以参考我分享的例子。

```python
import os
import subprocess

# 1. 使用 PySceneDetect 生成 EDL 文件
# ==========================================
def generate_edl(video_path, output_dir):
    # 使用 subprocess 调用 scenedetect 命令行工具
    # detect-content 使用内容感知算法，save-edl 导出达芬奇支持的 EDL
    print(f"正在分析视频: {video_path}...")
    subprocess.run([
        "scenedetect", 
        "-i", video_path, 
        "detect-content", 
        "save-edl", "-o", output_dir
    ], check=True)
    
    # 获取生成的 edl 文件名 (通常是 视频名.edl)
    base_name = os.path.splitext(os.path.basename(video_path))[0]
    edl_path = os.path.join(output_dir, f"{base_name}.edl")
    return edl_path



# 主程序执行
# ==========================================
if __name__ == "__main__":
    VIDEO_FILE = input("请输入视频文件路径: ")
    OUTPUT_FOLDER = input("请输入输出文件夹路径: ")
    
    # 执行流程
    edl_file = generate_edl(VIDEO_FILE, OUTPUT_FOLDER)
```

这个程序会输出一个EDL文件，让我们来看看文件长什么样——

```json
* CREATED WITH PYSCENEDETECT 0.6.7.1
TITLE: test
FCM: NON-DROP FRAME

001  AX V     C        00:00:00:00 00:00:02:12 00:00:00:00 00:00:02:12
002  AX V     C        00:00:02:12 00:00:05:00 00:00:02:12 00:00:05:00
```

我们可以发现，它记录了剪切的时间点

选中视频，按下control+shift+I，选中EDL，创建一个新的时间线，就可以把剪切数据导进来。
