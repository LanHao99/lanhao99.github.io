Links 中转站

这里现在不仅是导航页，也是一个可本地维护的小型页面目录。你可以手动添加新页面、编辑现有条目、调整顺序，浏览器会自动记住这些改动。

0 当前条目数
3 内置子页面
Local 本地保存模式

使用说明

新添加或编辑后的导航项会保存在当前浏览器的 LocalStorage 里，不会自动提交到仓库。适合你先在本机维护目录结构，再决定哪些链接需要正式写回站点。

内置页面：医学院课表、WebGal 教程、Twine 教程。你也可以继续挂接其他仓库页、学习页或工具页。

页面目录

点击卡片进入页面。这里显示的是当前生效的全部条目顺序。

提示：如果你换浏览器或清理站点数据，本地添加的链接不会自动同步过去。

添加新页面

填写标题、说明和地址即可加入首页目录。编辑模式下保存会覆盖当前选中的条目。

建议：仓库内页面优先使用相对路径或站内路径，例如 ./twine/、./medical-schedule.html、/Novel/。

排序与维护

使用上移、下移调整顺序；编辑会把数据带回左侧表单；删除会立刻从当前浏览器配置中移除。
